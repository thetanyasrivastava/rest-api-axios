# rest-api-axios

# 🚀 REST API with Axios

## 📌 Project Overview

This project demonstrates the implementation of RESTful API operations using Axios in a Node.js environment. It covers all major HTTP methods: GET, POST, PUT, PATCH, and DELETE.

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* Axios
* JavaScript

---

## 📂 Features

* ✅ GET request to fetch data
* ✅ POST request to create data
* ✅ PUT request to update entire resource
* ✅ PATCH request to partially update resource
* ✅ DELETE request to remove data
* ✅ Error handling for API calls
* ✅ Environment variable management using `.env`

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo.git
```

2. Navigate to the project folder:

```
cd your-repo
```

3. Install dependencies:

```
npm install
```

4. Create a `.env` file and add:

```
PORT=5000
API_URL=your_api_url_here
```

5. Run the server:

```
node index.js
```

---

## 📬 API Endpoints

| Method | Endpoint      | Description         |
| ------ | ------------- | ------------------- |
| GET    | /api/data     | Fetch data          |
| POST   | /api/data     | Create data         |
| PUT    | /api/data/:id | Update full data    |
| PATCH  | /api/data/:id | Update partial data |
| DELETE | /api/data/:id | Delete data         |
